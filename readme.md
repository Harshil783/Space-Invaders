# Space Invaders

Space Invaders is a popular game, here its implemented in python.

![](Space_invaders.png)

## Installation

OS X & Linux:

```bash
pip install -r requirements.txt
```

Windows:

```cmd
pip install -r requirements.txt
```

## Usage example

Run the main.pyw file to play the game
